Aula 1 - Elementos, [strong] e [em]
  • Elementos: É o elemento raiz de todo documento HTML, ele é caracterizado por uma tag de abertura seguida por outra de fechamento.
  • [Strong]: semanticamente(traz uma importancia para o texto dentro dessa tag), visualmente(estiliza o texto para negrito).
  • [em]: semanticamente(traz uma importancia para o texto dentro dessa tag), visualmente(estiliza o texto para italico).
Aula 2 - [!], (lang), [DOCTYPE],[title], [head], [body].
  • [!]: usado no Visual studio code para autocompletar o codigo fonte base do HTML5.
  • (lang): atributo usado no elemento [html] para definir a linguagem nativa do documento.
  • [DOCTYPE]: elemento usado para definir que o documento esta na ultima versão do HTML, a versão 5.
  • [title]: elemento que usamos para definir um titulo ao documento que sera visto nas abas dos navegadores.
  • [head]: todos elementos dentro dele são de configuração relacionadas ao documento.
  • [body]: todos os elementos dentro dele codificados compoem visualmente a pagina redenizada pelos navegadores.
Aula 3 - Atributos.
Quando escrevemos um Atributo dentro de um elemento, dizemos que queremos atribuir/adicionar alguma informação/valor aquele Elemento

Aula 4 - Navegador DevTools, DOM
  • Navegador DevTools: Ferramenta de inspecionar que auxilia os desenvolvedores.
  • DOM: Estrutura de arvore dos elementos pais, filhos e irmãos.
Aula 5 - Link, listas e imagens.
  • Link: Usamos o elemento [a] para adicionar links na pagina, no atributo [href] é onde adicionaos o endereço.
  • listas: Usamos o elemento [ul,ol,dl] para adicionar listas ao site, onde usamos o elemento[li] para incluir itens.
  • imagens: Usamos o elemento [img] para adicionar imagens ao site, no atributo [src] é onde adicionamos a origem da imagem.
Aula 6 - [Table], [thead], [tbody], [tr], [th], [td].
  • [table]: Elemento pai responsavel pela tabela.
  • [thead]: Elemento filho de [table] que representa o cabeçalho da tabela.
  • [tbody]: Elemento filho de [table] que representa o conteudo da tabela.
  • [tr]: Elemento filho de [thead], [tbody] que representa uma linha.
  • [th]: Elemento filho de [tr], que represneta uma celula do elemento[thead].
  • [td]: Elemento filho de [tr], que representa uma celula do elemento [ebody].
Aula 7 - [form], (action), [input], (type), (placeholder), [label], email, password, text, [button], submit.
  • [form]: elemento pai responsavel pelo formulario.
  • (action): atributo que recebe o valor em link onde recebera os dados do formulario.
  • [input]: elemento responsavel pela entrada e envio de dados.
  • (type): atributo que recebe o valor que caracterizara o elemento [action].
  • (placeholder): atributo que recebe o valor opcional que exibira uma mensagem no campo vazio.
  • [label]: elemento que define o titulo de cada item no formulario.
  • email: valor que é definido no atributo (type) e recebe dados de email.
  • password: valor que é definido no atributo (type) e recebe dados de senha.
  • text: valor que é definido no atributo (type) e recebe dados de texto.
  • [button]: elemento responsavel pelo botão clicavel.
  • submit: valor que é definido no atributo (type) e envia dados ao link definido no atributo (action) dentro do elemento [form].
