Aula 1 - Link css, arquivo css, comentarios, [seletor], declaração, (propriedade), {valor}, margem global.
  • Link css: link rel="stylesheet" href="local do .css". Ele é responsavel por relacionar o documento HTML5 ao arquivo de estilos do CSS3.
  • Arquivo css: style.css. Ele é responsavel por conter toda estilização que vai ser aplicada ao documento HTML5 mediante linkamento.
  • Comentarios: !-- comente -- . Usado para comentar trechos do documento HTML5. /* comente */ usado para comentar trechos do documento CSS3.
  • [seletor]: Elemento escolhido para receber a estilização.
  • declaração: [seletor] { declaração }, espaço entre chaves que recebe (propriedades) e {valores}.
  • (propriedade): color: black , pode ser definido por cor, size etc, que recebe valores variaveis dependendo da necessidade da estilização.
  • {valor}: Definido dentro da (propriedade), pode ser variavel por cor, size etc, dependendo da necessidade da estilização.
  • margem global: margin: 0px, padding: 0px, box-sizing: border-box. usada para definir uma margem padrão para todos navegadores.
